#유저파일/설정 

nfs 설정 파일 - 어떤 디렉터리를 **어떤 호스트/네트워크에 마운트 허용할지**가 적혀 있음.

```
/home/nagox   10.0.0.0/24(rw,sync,no_root_squash)
```

👉 여기서 얻을 수 있는 정보:

1. **내부 네트워크 범위** (`10.0.0.0/24`)
    
    - 내부망 IP 대역을 알 수 있음.
        
2. **마운트 가능 디렉터리** (`/home/nagox`)
    
    - 해당 디렉토리에 중요한 데이터나 키가 있을 수 있음.
        
3. **취약한 옵션**:
    
    - `no_root_squash` → 클라이언트에서 root 권한이 그대로 유지됨 → 원격 root 권한 획득 가능.
