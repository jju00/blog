[[웹 취약점 진단]]

>sqli하다가 안 된다? -> 자동화도구 사용
>gobuster하는데 디렉터리 안 나온다? -> 자동화도구(nikto) 사용

***이런 식으로 뭔가 하다가 안 되면 자동화 한 번 시도 or 처음부터 자동화 돌려놓기***


sqli 자동화 도구: sqlmap

1. GET 요청 (URL로 파라미터 보내는)
2. POST 요청 (로그인 폼 등 http body에 데이터 보내는)

에 따라 명령어 다름. 2번 같은 경우
```
sqlmap -u "URL" --cookie "쿠키값"
```
으로 취약한 파라미터가 있는지 없는지 식별 후 본격 공격하면 됨
